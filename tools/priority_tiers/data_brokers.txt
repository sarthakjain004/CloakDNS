# Data brokers and cross-device identity graphs.
#
# These are the "back-end" of the tracking ecosystem — they ingest
# online IDs, marry them with offline purchase / loyalty / DMV /
# voter-roll data, and feed the result back as targeting attributes.
# Blocking these severs the link between online activity and offline
# identity.
#
# Sources:
#   - "Guarding Digital Privacy" (arXiv 2025) — data broker section
#   - Sanchez-Rola et al. "Web Watching You" (2017)
#   - Englehardt & Narayanan (CCS 2016) — cross-domain ID syncing graph
#   - Federal Trade Commission "Data Brokers" report (2014, 2025 updates)
