# Cookie-syncing hubs and major SSP/DSP/exchange domains.
#
# Extends CLAUDE.md tier1+2 (in blocklists/tier1.txt) with the
# next-tier domains identified by Englehardt & Narayanan's CCS 2016
# 1M-site OpenWPM census as significant cookie-syncing partners or
# third-party trackers reaching >0.1% of the top 1M.
#
# A "cookie syncing" event is when one tracker writes its ID into a
# request URL routed via another tracker's domain so the second can
# read it. Englehardt 2016 Table 4 showed adverticum.net's ID appearing
# on 752 of the top 1k sites via 82 syncing partners. Killing the hubs
# kills the graph.
#
# Sources:
#   - Englehardt & Narayanan, "Online Tracking: A 1M-Site Census" (CCS 2016)
#   - Acar et al., "The Web Never Forgets" (CCS 2014)
#   - Roesner et al., "Detecting and Defending Against Third-Party Tracking" (NSDI 2012)
