# * facebook.com / yahoo.com / bing.com / google.com / google.co.in /
#   snapchat.com / linkedin.com / quora.com / temu.com / indiatimes.com /
#   flixcart.com — first-party for many users; only specific subdomains
#   blocked above where they're clearly adtech.

#   Financial Times static.

#   GDPR/CCPA banner; blocking it tends to *break* the consent flow rather
#   than improve privacy.

#   Add it if you want stricter blocking.
#
# * AWS / Akamai / Cloudfront / GCP CDN domains and *.run.app — shared
#   infrastructure that hosts adtech AND legitimate content; CNAME
#   uncloaking handles the adtech cases at runtime.
